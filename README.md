# AVATAR

## Setup
after cloning, run:

git submodule update --init --recursive

to get external modules (currently the hmibuild system).

## Components

### ASAPRealizer
Compile & run the starter by runnig these commands in ./java/AVATARStarters/:

ant resolve
ant compile
ant main (select AsapAvatarStarter class)
ant run
